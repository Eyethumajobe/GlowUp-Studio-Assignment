# GlowUp-Studio-Assignment
Eyethu Hair Salon
1.	Introduction 
GlowUp Studio is a beauty and wellness business. The website was created to promote the business online and help customers learn about the services offered.

2.	Business Overview
Business Name: GlowUp Studio
Industry: Beauty and Wellness
Target Market: Teenagers, young adults and working professionals.

3.	 Website Objectives
• Promote beauty services online.
• Attract new customers.
• Share business information.
• Make it easy for customers to contact the business.

4.	 Website Features
• Home Page
• About Us Page
• Services Page
• Contact Form
• Image Slideshow
• Social Media Links
• Mobile-Friendly Design

5.	 Technologies Used
The website was developed using HTML5, CSS3, Font Awesome icons and media queries for responsive design.

6.	 Benefits of the Website
• Available 24 hours a day.
• Reaches more customers.
• Improves the business image.
• Provides easy communication with customers.

7.	 Conclusion
The GlowUp Studio website helps the business grow its online presence and connect with customers. It is simple, attractive and easy to use on both computers and mobile phones.
